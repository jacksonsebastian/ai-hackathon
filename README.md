# AI Interviewer Agent System

Build a hackathon-winning AI Interviewer Agent Platform for the TCS AI Hackathon that conducts technical, behavioral, and coding interviews using multi-agent orchestration, RAG pipelines, fine-tuned LLMs, and GPU-accelerated inference.

## Quickstart
1. Rename `.env.example` to `.env`
2. Install dependencies: `make install`
3. Run: `make run`
